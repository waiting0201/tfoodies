為什麼會斷在這裡？兩個原因

原因 A：廣告把「想買的冷流量」帶到了「文章頁」

文章是用來「養」客人、做 SEO 的，不是用來成交第一次看到你的人。冷流量需要的是「這是什麼、多少錢、為什麼安心、現在買有什麼好處」——文章給不了，他就走了。

原因 B：文章看完，沒有一條路通往「買」

最可惜的是食譜：讀者看完想煮，卻找不到食材在哪買。「想煮 → 需要食材 → 能下單」這條最自然的路，現在是斷的，人就流失了。

食譜是食品電商的金礦——看食譜的人本來就「準備要買食材」。把這條橋接上，連免費的自然搜尋流量都能變訂單。

修法 A　把廣告的落地頁換掉

想「立刻有人買」的廣告，不要連到文章，連到一個明星商品頁＋首購優惠。

  • 挑 1 個最有把握的商品當主打，廣告直接連到它的商品頁。
  • 頁面一眼看懂：是什麼、多少錢、為什麼安心（檢驗/來源）、現在買有優惠。
  • 首購折扣碼直接顯示出來，不要藏在角落。
  • 文章留給 SEO 與自然流量，別再拿文章扛成交。

修法 B　把文章變成「會帶貨」的文章

這是你最強的資產。讓看文章的人（含免費的自然流量）也能順手下單：

  • 食譜頁底部放「本食譜用到的食材 → 一鍵加入購物車」。
  • 每篇文章結尾放 1–3 個相關商品卡，按了就能買。
  • 文章中段放一句行動呼籲（CTA）：「想試試？馬上備齊食材 →」。

💡 好消息：你的 store 綠誌詳情頁本來就有「相關商品/食譜」區塊，食譜頁很可能只是還沒接上——補起來會比想像中快。

03本週行動清單與成功指標

不用一次做完所有事。照下面勾，這週就能動手，兩週後用一個數字判斷有沒有效。

✅ 本週行動清單

  • 選定 1 個明星商品，檢查它的商品頁有沒有清楚寫「是什麼・價格・安心理由・優惠」。
  • 設定一組首購折扣碼，並讓它在頁面上顯眼。
  • 把現有廣告的落地連結改成那個商品頁（不要再連文章）。
  • 選 3 篇最熱門的食譜/文章，先手動在結尾放上「相關商品 / 加入購物車」連結。
  • （技術）盤點食譜詳情頁，把「食材一鍵加入購物車」區塊接上。

📊 怎麼知道有沒有效（看這一個數字）

加入購物車率加購人數 ÷ 進站人數
這是這次的關鍵指標
從 ≈0%現況：幾乎沒人加購
→ 目標 3%+2 週後的第一個里程碑

邏輯很簡單：先讓「加入購物車」這關活起來，有人加購，後面結帳、付款才有東西可優化。只要加購率從 0 開始往上走，就代表破洞補對了，這時再加廣告預算放大才划算。

一句話總結

現在的問題不是「沒人來」，是「來了沒東西可買 / 不知道怎麼買」。把廣告帶到能買的頁面，把文章接上購物車——讓加購率動起來，訂單就會跟著來。
